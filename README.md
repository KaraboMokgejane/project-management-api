Advanced Task & Project Management API

A secure, production-ready RESTful API designed to streamline project tracking, task assignment, and team collaboration. Built using **Java** and **Spring Boot**, this project follows clean architecture principles, robust exception handling, and enterprise-grade design patterns.

---

 Key Features
* **Full CRUD Operations:** Seamlessly create, read, update, and delete projects and tasks.
* **Persistent Data Layer:** Utilizes Spring Data JPA for efficient database communication and object-relational mapping.
* **Automated Auditing:** Automatically tracks timestamps for task creation, status modifications, and deadlines.
* **Scalable Architecture:** Structured into Controller, Service, Repository, and Model layers to ensure separation of concerns.

---

Tech Stack & Tools
* **Backend:** Java 17+, Spring Boot (Web, Data JPA)
* **Database:** H2 In-Memory Database (configured for development/testing)
* **Build Tool:** Maven
* **Libraries:** Lombok (boilerplate reduction)
